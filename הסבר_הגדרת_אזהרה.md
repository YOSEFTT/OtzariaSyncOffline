# הסבר על הגדרת "אל תזכיר עוד פעם" 📝

## איפה ההגדרה נשמרת? 💾

כשאתה בוחר באפשרות **"אל תזכיר לי עוד פעם"** בהודעת האזהרה על גירסת ספרייה 53, ההגדרה נשמרת ב-**2 מקומות**:

### 1. קובץ מקומי: `sync_state.json` 📁
- **מיקום**: בתיקיית התוכנה (ליד הקובץ `otzaria_sync_offline.py` או ה-EXE)
- **יתרון**: ניתן להעתיק את הקובץ למחשב אחר והבחירה תישמר!
- **שם השדה**: `dont_show_version_53_warning`

### 2. Windows Registry (QSettings) 🔧
- **מיקום**: `HKEY_CURRENT_USER\Software\OtzariaSync\Settings`
- **יתרון**: שמירה מקומית למחשב הנוכחי
- **שם המפתח**: `dont_show_version_53_warning`

## איך לבטל את הבחירה? 🔄

### דרך 1: מחיקת השדה מהקובץ (מומלץ) ✏️

1. פתח את הקובץ `sync_state.json` בעורך טקסט
2. מצא את השורה:
   ```json
   "dont_show_version_53_warning": true,
   ```
3. **מחק** את השורה הזו (או שנה ל-`false`)
4. שמור את הקובץ
5. הפעל מחדש את התוכנה - ההודעה תופיע שוב!

### דרך 2: מחיקת הקובץ כולו 🗑️

פשוט מחק את הקובץ `sync_state.json` - התוכנה תיצור אותו מחדש בפעם הבאה.

### דרך 3: דרך Registry Editor 🔧

1. לחץ `Win + R`
2. הקלד: `regedit`
3. נווט ל: `HKEY_CURRENT_USER\Software\OtzariaSync\Settings`
4. מחק את המפתח `dont_show_version_53_warning`

### דרך 4: דרך Python Console 🐍

```python
from PyQt6.QtCore import QSettings
import json
import os

# מחיקה מ-QSettings
settings = QSettings("OtzariaSync", "Settings")
settings.remove("dont_show_version_53_warning")

# מחיקה מהקובץ המקומי
if os.path.exists("sync_state.json"):
    with open("sync_state.json", "r", encoding="utf-8") as f:
        state = json.load(f)
    
    if "dont_show_version_53_warning" in state:
        del state["dont_show_version_53_warning"]
        
        with open("sync_state.json", "w", encoding="utf-8") as f:
            json.dump(state, f, indent=2, ensure_ascii=False)
        
        print("ההגדרה נמחקה מהקובץ!")

print("ההגדרה נמחקה!")
```

## העתקה למחשב אחר 💻➡️💻

אם אתה רוצה שהבחירה תישמר גם במחשב אחר:

1. העתק את הקובץ `sync_state.json` מהמחשב הראשון
2. הדבק אותו בתיקיית התוכנה במחשב השני
3. הפעל את התוכנה - ההגדרה תיטען אוטומטית!

## איך זה עובד טכנית? 🤓

התוכנה קודם בודקת את הקובץ המקומי `sync_state.json`, ואם לא מוצאת שם את ההגדרה, היא בודקת ב-Registry (QSettings).

זה מאפשר:
- ✅ נייד בין מחשבים (דרך הקובץ)
- ✅ תאימות לאחור (דרך Registry)
- ✅ גיבוי אוטומטי (שני מקומות)
